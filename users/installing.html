Installing¶

pyregion can be downloaded from the following page.

  • Download

However, pyregion is a personal hobby project and still in development (likely to be buggy). And it is recommended that you fetch the source code by cloning my git repository for any recent bug fix.

# git clone git://github.com/leejjoon/pyregion.git
# cd pyregion
# python setup.py install

Dependencies¶

Requirements

pyregion is based on pyparsing module. You also requires Kapteyn package or pywcs module (you need only one of them) for coordinate conversion. Displaying regions is supported for matplotlib. Some example uses pywcsgrid2.
